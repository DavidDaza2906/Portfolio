Retail Image Classifier

Deep Learning CNN for retail clothing classification. MobileNetV3-Small fine-tuned on 17.7K real studio photos from DeepFashion InShop. 78.5% accuracy across 12 categories. Exported to ONNX with embedded weights (6.1MB) and running 100% in-browser inference with ONNX Runtime Web.

01 Overview

Image classifier trained on real retail product photos using transfer learning with MobileNetV3-Small. The model handles the challenging conditions of studio product photography: varying lighting, different angles, and diverse clothing styles. Exported to ONNX format with weights embedded in a single 6.1MB file for instant browser inference.

03 Training Pipeline

Step 1

Dataset

DeepFashion InShop filtered to 17.7K images across 12 categories. Real studio photos with diverse poses, lighting, and backgrounds.

Step 2

Transfer Learning

MobileNetV3-Small pre-trained on ImageNet. Classifier head replaced for 12 retail categories. 1.5M trainable parameters.

Step 3

Training

8 epochs with Mixup augmentation, Label Smoothing, and OneCycleLR scheduler. Trained on AMD RX 6600 XT via ROCm.

Step 4

Export

ONNX opset 18 with embedded weights. Single 6.1MB file. No external dependencies for inference.

Step 5

Inference

100% browser-side via ONNX Runtime Web. Preprocessing pipeline visible: resize center-crop normalize CNN classify.
